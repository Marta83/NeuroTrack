# NeuroTrack

Base de proyecto Flutter (Material 3) con Firebase, Riverpod y go_router.

## Pendiente en tu entorno

1. Instalar Flutter SDK.
2. Ejecutar `flutter create .` para generar carpetas nativas (`ios/`, `android/`, `web/`).
3. Ejecutar `flutter pub get`.
4. Configurar Firebase real con FlutterFire (`flutterfire configure`) y reemplazar `lib/firebase_options.dart`.
